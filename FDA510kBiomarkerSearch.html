FDA 510(k) Biomarker Search
openFDA Device Clearance Lookup
Uses the public openFDA device 510(k) API. No key: 1,000 requests/day. Add a free key for 120,000/day.
LDT search calls a small proxy Worker (NY's LDT site has no CORS support, so this tool can't reach it directly) — see worker/ldt-proxy.js in the repo for the code and deploy instructions.
Biomarkers
Terms are matched against device name and 510(k) summary text (openFDA has no dedicated "biomarker" field). Greek letters (α, β, …) are automatically spelled out (alpha, beta, …). If an exact phrase finds nothing, the search automatically widens — first dropping word order, then the Ig-class suffix — and flags the result as Broad match / Antigen-only match so you know it's less precise. "Anti-" is never dropped when the term has it — it distinguishes an antibody test from the antigen itself, so a real match must still contain it.
Terms are matched against the analyte/test name in the NY State Wadsworth Center CLEP "Approved Laboratory Developed Tests" database (any-word match, same as the site's own default). Requires an LDT proxy Worker URL configured in Settings (gear icon).
